# 🚀 Map-Based Shortest Path Finder using Dijkstra’s Algorithm

## 📌 Project Overview

The **Map-Based Shortest Path Finder** is an interactive application that calculates the shortest path between two locations using **Dijkstra’s Algorithm**.

This project demonstrates how graph-based algorithms are used in real-world applications such as **GPS navigation systems, route optimization, and network routing**.

---

## 🎯 Objectives

* Implement Dijkstra’s Algorithm efficiently
* Visualize shortest path in a graph/map
* Provide user-friendly interaction
* Understand real-world applications of graph algorithms

---

## 🛠️ Technologies Used

* **Programming Language:** Python / C++
* **GUI:** Tkinter / HTML, CSS, JavaScript
* **Libraries:** heapq (Priority Queue)
* **Concepts:**

  * Graph Data Structure
  * Dijkstra’s Algorithm
  * Greedy Algorithm

---

## ⚙️ How It Works

1. Create or load a graph with nodes and edges
2. Assign weights (distance/cost) to edges
3. Select source and destination nodes
4. Algorithm computes shortest path
5. Output is displayed visually and numerically

---

## 🧠 Algorithm Used

**Dijkstra’s Algorithm**

* Starts from source node
* Assigns minimum distance to each node
* Uses a priority queue (min-heap)
* Finds the shortest path efficiently
